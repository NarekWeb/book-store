# Bookstore API (Laravel 12 + Docker)

Bookstore is a Laravel 12 application (Dockerized with **Laravel Sail**) that demonstrates:

- REST API for a simple book/rental system
- Redis-backed queues
- **Laravel Horizon** for queue monitoring
- **Laravel Telescope** for debugging
- Custom install command: `app:install`

---

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/NarekWeb/book-store.git
cd book-store
```

### 2. Run the project with docker

### 3. Run app
```bash
php artisan app:install
```
